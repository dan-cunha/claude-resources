# claude-resources

Claude Code plugins and resources by Dan da Cunha.

## Marketplace

### Add the marketplace

```bash
claude /plugin marketplace add https://github.com/dan-cunha/claude-resources/marketplace
```

### Available plugins

| Plugin | Description |
|--------|-------------|
| **reprompt** | Session continuation prompt generator — captures decisions, state, and references for resuming in a new session |

### Install a plugin

```bash
claude /plugin install reprompt@ddc-claude-plugins
```

### Update plugins

```bash
claude /plugin marketplace update ddc-claude-plugins
```
